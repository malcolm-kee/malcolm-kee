## React Compiler - build time optimization

[React Compiler](https://react.dev/learn/react-compiler) is a build-time tool that automatically optimizes your React app.

### The simplest possible case

A component with no props and constant JSX.

The compiler wraps the `<div>` in a cache slot guarded by the `IS_UNINITIALIZED` sentinel, so the JSX element is created exactly once and reused on every subsequent render.
