## Why Use Typescript for React

- Make code more maintainable
- Better job prospect

## This Workshop Covers

1. Typescript fundamentals
1. How to slowly transition your React codebase from JavaScript to Typescript
1. Common use cases when using Typescript with React

## This Workshop Does Not Cover

1. React fundamentals (for that, refer to another workshop, [Introduction to React v2](/intro-to-react-js-v2))
1. Modern JavaScript (for that, refer to another workshop, [JavaScript: The React Parts](/js-the-react-parts))
