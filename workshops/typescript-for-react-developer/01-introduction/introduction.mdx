## Why Learn TypeScript as React Developer

1. Make code more maintainable

   - code is self-documented
   - easier to refactor

1. Better job prospect

## This Workshop Covers

1. TypeScript fundamentals
1. How to slowly migrate your React codebase from JavaScript to TypeScript
1. Common coding pattern when using TypeScript with React

## This Workshop Does Not Cover

1. React fundamentals (if you interested, refer to another workshop, [Introduction to React v2](/intro-to-react-js-v2))
1. Modern JavaScript (for that, refer to another workshop, [JavaScript: The React Parts](/js-the-react-parts))
